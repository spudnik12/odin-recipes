Odin Project: Recipes Guidelines

Project should meet the following characteristics:

  • Directory contains an index page with:
    • One leading <h1> heading
    • links to 3 recipes in its sub-directory
  • Each recipe contains in the following order:
    • One leading <h1> heading
    • An image of the completed recipe
    • <h2> heading containing "Description"
    • 1 - 2 paragraphs describing the recipe
    • <h1> heading containing "Ingredients"
    • Unordered listing of each ingredient using <ul>
    • <h1> heading containing "Steps"
    • Ordered listing of the recipe using <ol>
